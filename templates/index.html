Install

Download JS.py using git clone or by downloading the zip file from git hub.

git clone https://github.com/DenerosArmy/pythonscript.git

Run an Example

Open your terminal and navigate to the pythonscript folder then enter this command

Please note that you must use Python 2.7

python js.py min_1.py

Output:

var py = require("./py");
console.log("Test in_1");;
foo = py.def (py.kwargs, {"y":1}, function  (x, y, kwargs) {
    console.log (x + y);
});
foo ([], {"x":1, "y":2});
console.log("End test in_1");;

Formatting

Add the following markup

from pythonscript import js
js('var py = require("./py")')
js('console.log("Test in_1");')
def foo(x, y=1, **kwargs):
    print x+y
foo(**{'x': 1, 'y': 2})
js('console.log("End test in_1");')

Usage

Save the generated JS file and use the <script src='yourpy.js'> tag (in your html HEAD section), then use the JS like any other function. You can even use alert functions (see the docs)

<script>
js.alert('Goodbye JS')
</script>

Requiremetnds

What you need:
Python 2.7
Optional: Flask for compiling on a web server and displaying the output (in html). Take the compiled JS and use it like any other script. The long term goal for this project is to be able to do something like this

<script src="script.py" type="text/python">
js.alert('Goodbye JS')
</script>
